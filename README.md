# Quinn Redwoods Portfolio

Live at [qredwoods.com](https://www.qredwoods.com)

An html page with css and minimal javascript

Fonts: DM Sans for headings, Inter for body

Notable detail: the styling of "social-band" section

## Run

From the project  root:

```bash
python3 -m http.server 8000```

Then open `http://localhost:8000:`
